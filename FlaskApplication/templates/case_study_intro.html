Dataset Overview

The dataset titled "Employability Classification of Over 70,000 Job Applicants" contains a comprehensive collection of information regarding job applicants and their respective employability scores.

This dataset has been compiled to assist organizations and recruiters in evaluating the suitability of candidates for various employment opportunities. By utilizing machine learning techniques, it aims to:

  • Provide valuable insights into factors influencing employability
  • Enhance the efficiency of the hiring process

Goal

The aim of this study is to conduct a deeper analysis on this dataset regarding job applicants. The goal is to predict employability based on applicants' characteristics/skills present in the feature HaveWorkedWith (e.g., 'C++;Python;Git;PostgreSQL').

Workflow:

  1. Data transformations:
    • TF-IDF vectorization for text data conversion
    • Features Selection
    • Model Selection
    • Model Tuning
  2. Model performance analysis:
    • Features Importance
    • Learning curve
    • Confusion Matrices
    • Precision-Recall Curve
    • ROC curve

For more details, check the Jupyter notebook available on GitHub.
